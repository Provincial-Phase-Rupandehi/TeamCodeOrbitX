## ‚ö†Ô∏è Common Demo Issues & Solutions

### Issue: Voice Recognition Not Working
**Solution:** 
- Check browser (Chrome/Edge work best)
- Enable microphone permission
- Have backup: Show screenshots

### Issue: QR Code Scanner Not Working
**Solution:**
- Use manual input option
- Have pre-generated QR code ready
- Show screenshot of working scanner

### Issue: Offline Mode Not Triggering
**Solution:**
- Use browser DevTools Network throttling
- Show screenshots
- Explain the feature conceptually

### Issue: Predictions Empty
**Solution:**
- Explain it needs historical data
- Show the prediction logic
- Display sample predictions
